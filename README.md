# QuantumOps AI

**AI-Powered Monitoring & Automation for Proof-of-Stake Networks**

QuantumOps AI is an intelligent operations platform for blockchain validators.  
It combines:

- **Monitoring** – real-time validator and node health
- **Automation** – self-healing and upgrade actions
- **AI Assistance** – explanations, recommendations, and anomaly analysis

## Documentation

Project documentation is located in the `/docs` folder:

- `/docs/product` – product vision, roadmap, features  
- `/docs/infra` – validator & infra setup  
- `/docs/ai` – AI assistant design  
- `/docs/automation` – automation flows  
- `/docs/api` – API design
