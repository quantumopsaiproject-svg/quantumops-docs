# Infrastructure Playbook

## Validator Setup
- Validator server (4–8 CPU, 16–32 GB RAM)
- Sentry node(s)
- Prometheus exporter enabled
- RPC availability
- Firewall rules
- State-sync & snapshot usage

## Monitoring Setup
- Node exporter
- Tendermint/Golang metrics
- Prometheus configuration
- Grafana dashboard basics
- Alerting rules (offline, missed blocks)

## Automation Setup
- Systemd restart scripts
- Peer reset
- Safe upgrade flow
- Backup & restore workflows
